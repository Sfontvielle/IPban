Автоматическая загрузка через fetch() не сработала (это нормально, если страница открыта как локальный файл — file:// — как на скриншотах IPBan под Windows: браузер такие запросы блокирует). Нажмите «➕ Добавить папку с логами» вверху страницы и выберите папку целиком — можно нажать эту кнопку несколько раз для разных папок (например, отдельно папку IPBan и отдельно папку RadiusProxy) — все они объединяются в одну картину. Дашборд сам найдёт в каждой папке файлы логов (какое бы имя они ни носили — файлы опознаются по содержимому, по первым строкам с датой, а не по конкретному имени) и файл .sqlite, объединит логи по их реальным меткам времени и покажет свежий день по умолчанию (другой день можно выбрать в поле «Дата лога» вверху). Браузер запомнит доступ к папкам, так что «Обновить сейчас» и автообновление продолжат работать без повторного выбора. Ничего никуда не загружается — всё остаётся в браузере.
